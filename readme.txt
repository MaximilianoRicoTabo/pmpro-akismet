Connect your membership site to Akismet to protect yourself from checkout spam.

== Description ==

This plugin integrates the Akismet Plugin with Paid Memberships Pro checkouts to protect your membership site’s checkout process from spam. 
Similar to how Akismet filters comment spam, this integration will check the submitted email address against Akismet’s world-class spam filters.

== Installation ==

= Prerequisites =
1. You must have Paid Memberships Pro installed and activated on your site.

= Download, Install and Activate! =
1. Download the latest version of the plugin.
1. Unzip the downloaded file to your computer.
1. Upload the /pmpro-akismet/ directory to the /wp-content/plugins/ directory of your site.
1. Activate the plugin through the 'Plugins' menu in WordPress.

= How to Use =
1. Install Akismet WordPress plugin and configure to your liking.
1. PMPro Akismet will automatically detect spam at checkout and prevent submissions if they are considered spam.

View full documentation at: https://www.paidmembershipspro.com/add-ons/pmpro-akismet/
